# 📂 Manejo de Ficheros

Aprenderemos a interactuar con el sistema de archivos utilizando Java I/O y NIO.

## Contenidos
- La clase `File`.
- Streams de entrada y salida (FileInputStream/FileOutputStream).
- Serialización de objetos.
- Acceso aleatorio con `RandomAccessFile`.
