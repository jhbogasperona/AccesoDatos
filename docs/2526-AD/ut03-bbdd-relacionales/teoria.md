# 🗄️ JDBC (Java Database Connectivity)

JDBC es la API estándar de Java para interactuar con bases de datos relacionales.

## Componentes Clave
- `DriverManager`: Gestiona los drivers.
- `Connection`: Representa la sesión con la BBDD.
- `Statement` / `PreparedStatement`: Ejecución de consultas.
- `ResultSet`: Contenedor de los resultados.
