# 📑 Lenguaje de Marcas y Sistemas de Gestión de Información (LMSGI)

¡Bienvenido al módulo de **LMSGI**! En este espacio exploraremos las tecnologías fundamentales que permiten estructurar, presentar e intercambiar información en el ecosistema digital moderno.

:::info ¿Qué es LMSGI?
Este módulo se centra en el estudio de los lenguajes de marcas (como HTML, XML y JSON) y cómo estos se utilizan para organizar la información de manera que sea legible tanto para humanos como para máquinas.
:::

---

## 🎯 Objetivos del Curso

Al finalizar este módulo, serás capaz de:
- **Estructurar documentos** utilizando lenguajes de marcas estándar.
- **Crear interfaces web** modernas y responsivas con HTML5 y CSS3.
- **Gestionar información** mediante lenguajes de intercambio de datos.
- **Añadir interactividad** básica a tus páginas utilizando JavaScript.
- **Validar y transformar** datos para asegurar su integridad.

## 🛠️ Tu "Stack" de Trabajo

Para este curso, te recomendamos configurar el siguiente entorno de desarrollo:

1.  **Editor de Código:** [Visual Studio Code](https://code.visualstudio.com/) (Imprescindible).
2.  **Navegador:** Chrome o Firefox (Developer Editions recomendadas).
3.  **Terminal:** Familiarízate con la consola de comandos.
4.  **Extensiones Pro:**
    *   `Live Server` (para previsualización en tiempo real).
    *   `Prettier` (para mantener un código limpio).
    *   `ESLint` (para detectar errores en JavaScript).

:::tip ¡Manos a la obra!
No tengas miedo de experimentar. La mejor forma de aprender es consultando la documentación oficial (MDN es tu mejor amiga) y practicando constantemente. 
:::
